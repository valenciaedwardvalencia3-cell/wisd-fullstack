# 🚀 WISD Fullstack Project

Proyecto fullstack desarrollado como parte de un sistema web que integra frontend y backend, permitiendo la gestión de datos mediante una API y una interfaz de usuario moderna.

---

## 📌 Descripción

**WISD Fullstack** es una aplicación web completa que conecta un frontend interactivo con un backend funcional.  
El sistema permite la comunicación entre cliente y servidor mediante servicios API REST, facilitando la gestión de información de manera eficiente.

Este proyecto está orientado al aprendizaje y práctica de desarrollo fullstack utilizando tecnologías modernas.

---

## 🛠️ Tecnologías utilizadas

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript
- Axios (consumo de API)

### Backend
- Node.js
- Express.js
- MongoDB (o la base de datos que estés usando)
- Mongoose
- API REST

## ⚙️ Instalación y ejecución

### 1. Clonar el repositorio
```bash
git clone https://github.com/valenciaedwardvalencia3-cell/wisd-fullstack.git

2. Backend
cd backend
npm install
npm start

3. Frontend
cd frontend
npm install
npm start

🔗 Conexión Frontend + Backend
El frontend consume la API del backend mediante peticiones HTTP (Axios/fetch), permitiendo:
Crear datos
Leer datos
Actualizar datos
Eliminar datos (CRUD)
